# Personal Portfolio
Hello!!! :D. This repository contains the source code for my personal portfolio website. Feel free to download it and check it out, or visit the github pages link. You can also fork it and use it as an inspiration.

## Goal
This project was not only to learn React, Typescript and Tailwind basic concepts, but also to have an online site to share more information about myself like current projects and working experience. 

# Technologies Used
- React
- Tailwind
- Vite
- Netlify

# Installation 
After forking and cloning the repo to your local machine just run the following commands
```
npm install
npm run dev
```
